# NVM Desk
> **NVM Desk** is an application that allows convenient usage of NVM on the desktop.

## Install
[Download](https://github.com/kdydesign/nvm-desk/releases) and install NVM-Desk.

⚠️ **NVM Desk** requires `NVM-Windows` to be installed. If `NVM-Windows` is not installed, please download and install it first from [nvm-windows](https://github.com/coreybutler/nvm-windows).

## Features
- Install the desired version of Node.js.
- Easily switch Node.js versions with a single click, without using commands.
